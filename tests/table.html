Info

Columns

Each column header represents a certain kind of material attributed to each resolved findspot. For example, a findspot with the kind "Tontafeln" means that a clay tablet was found at this location.

Rows

Each row header represents a population type attributed to each resolved toponym.

There are 3 categories for these population types - Troupes, Corps de metier, and Habitants.

Get Started

Each dot represents a toponym/findspot pair that have the given population and kind properties. To see more about each population and its related toponyms, click on a row.
